# SOLA ROOF — Solar-Ready Roofing Showcase

Scroll-animated website + material-breakdown cost estimator for a solar-ready roofing catalog (Kota Kinabalu, Sabah).

## Pages
- `index.html` — scroll-driven animation built from 120 video frames (WebP, desktop + mobile sets), with scroll-dwell pacing, ambient effects and a parallax gallery. Phone-friendly: smaller frame set, shorter scroll length and compact overlays on mobile.
- `dashboard.html` — interactive cost estimator with itemised material breakdown (roofing system, steel truss, insulation, gutter/flashing, installation allowance, solar PV, battery, carport), cost-split chart, solar payback estimate and WhatsApp lead capture.

## Run locally
```bash
python -m http.server 8080
# open http://localhost:8080
```

## Disclaimer
All prices are sample estimate ranges for budgeting discussion only — not a formal quotation.
